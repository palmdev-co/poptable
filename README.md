# poptable

A standard Power BI table with clickable rows that open a detail popup card.



PopTable is a Power BI custom visual that combines the familiarity of a standard table with an interactive popup detail card. Click any row to see additional fields in a clean overlay — perfect for dashboards where you need drill-down detail without navigating away.

FEATURES

• Standard table functionality — sortable columns, alternating row colors, sticky headers, keyboard navigation, and right-click context menu support.

• Row popup — add fields to the "Popup Details" data role and they appear in a formatted card when any row is clicked. The popup includes a header, labeled field/value pairs, a close button, and keyboard dismiss (Escape).

• Conditional formatting rules — up to 5 rules, each targeting a column by name. Choose from 13 operators (greater than, less than, equals, contains, starts with, between, is blank, and more). Format the background color, font color, bold, or italic — and optionally apply the rule to the entire row.

• Full format pane customization — control table font size, font family, row padding, header style (background, font color, bold, text transform), row style (background, alternate color, hover color), border and grid line settings, and all popup appearance options (width, height, corner radius, colors, dividers).

• Respects Power BI formatting — date and number formats set in the field well are applied correctly via the standard Power BI value formatter.

HOW TO USE

1. Add fields to the "Columns" data role to build your table.
2. Add fields to the "Popup Details" data role — these appear in the popup when a row is clicked.
3. Customize the look via the format pane. Set up conditional formatting rules by entering the column name, choosing an operator and value, and picking what to format.

IDEAL FOR

- Dashboards that need row-level detail without drillthrough pages
- Inventory, order, or ticket tables where you want quick access to extra fields
- Any scenario where the standard table works but you need a "show more" interaction

PopTable does not collect, store, or transmit any data. All processing happens within your Power BI environment. See the privacy policy for details.
